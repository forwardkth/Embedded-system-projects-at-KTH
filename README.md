# Embedded-system-projects-at-KTH

This repository includes 3 previous course projects based on different embedded system platform when I was a master student at KTH.

1. Car cruise control simulation (C language)
A car cruise control simulation program running under ucOS II on the Nios II CPU (Altera cyclone Dev kit).

2.cardio wireless EKG device (C language)
A handhold EKG device that support bluetooth communication with mobilephone
The device is implemented with MSP430 based board "wasa".

3.staff guarder (C language)
A mutisensor integration project. Implement a device that can guard your staff not to be moved by others.
